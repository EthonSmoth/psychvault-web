# Product Model

## Marketplace Sides

### Buyers

- browse resources and stores
- claim free downloads
- purchase paid resources
- leave reviews after purchase
- follow stores
- message creators

### Creators

- create and manage a store
- upload, price, archive, and publish resources
- manage previews and private downloadable files
- view sales, payouts, and analytics

### Admin

- review reports
- approve, reject, archive, or restore content
- moderate public marketplace visibility

## Resource Model

A resource can have:

- title
- short description
- full description
- pricing
- thumbnail
- preview assets
- private main download
- category
- tags
- moderation state
- review and sales aggregates

## Store Model

A store can have:

- name
- slug
- logo
- banner
- bio
- location
- publication status
- verification state
- owner relationship
