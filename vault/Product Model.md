# Product Model

## Marketplace Sides

### Buyers

- browse resources and stores
- claim free downloads
- purchase paid resources
- leave reviews after purchase
- follow stores
- message creators

### Creators

- create a store
- upload and price resources
- publish moderated listings
- view analytics and sales

### Admin

- review reports
- approve or reject content
- archive or restore public listings

## Resource Model

A resource can have:

- title
- short description
- full description
- pricing
- thumbnail
- preview assets
- private main download
- category
- tags
- moderation state
- review aggregates

## Store Model

A store can have:

- name
- slug
- logo
- banner
- bio
- location
- publication status
- verification state
